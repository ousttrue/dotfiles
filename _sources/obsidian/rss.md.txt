# Fever API

- [Fever API](https://miniflux.app/docs/fever.html)
- [Fever API implementation](https://freshrss.github.io/FreshRSS/en/developers/06_Fever_API.html)

- fetching categories
- fetching feeds
- fetching RSS items (new, favorites, unread, by_id, by_feed, by_category, since)
- fetching favicons
- setting read marker for item(s)
- setting starred marker for item(s)
- setting read marker for feed
- setting read marker for category
- supports FreshRSS extensions, which use the entry_before_display hook
