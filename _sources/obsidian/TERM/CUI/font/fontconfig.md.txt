# config
## fonts
`~/.fonts`
`/usr/local/share/fonts/`

## fonts.conf
`~/.config/fontconfig/fonts.conf`

## fc-cache
`update`
```
$ fc-cache -fv
# or ?
$ sudo fc-cache -fv
```

# query
## fc-list
[[xresource]]

```
xterm*faceName: Monoid:style=Regular:size=12:antialias=false
xterm*faceNameDoublesize: Yu Gothic:size=10
```

## fc-match
```
$ fc-match sans-serif
ipag.ttf: "IPAGothic" "Regular"
$ fc-match mono
ipag.ttf: "IPAGothic" "Regular"
```
