## partition 構成

> ストレージの第一パーティションを UEFI のシステムパーティション
> FAT32
> 512MB くらい？

```
/dev/sda1       511M  4.0K  511M   1% /boot/efi
```

| partition | size                           |
| --------- | ------------------------------ |
| EFI       | 512MB                          |
| SWAP      | MEMORY x 2 (for example: 32GB) |
| /         | 200GB                          |
