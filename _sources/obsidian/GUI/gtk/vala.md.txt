## entrypoint

```cpp
void main () {
    print ("hello, world\n");
}
```

C# とか Java ぽいが無意味。

```cpp
class Sample : Object {
  static void main(string[] args) {
    print ("hello, world\n");
  }
}
```

- 他の C library は、手書き(たぶん)
