# deps
`cmake.deps/CMakeLists.txt` による依存ライブラリの構築 

```sh
cmake -S cmake.deps -B .deps
```

と呼び出して `.deps` にビルドする。

## DEPS_INSTALL_DIR
- `.deps/usr` が本体構築時に `${DEPS_PREFIX}`  として参照される。

## cmake/Deps.cmake
```cmake
include(Deps)
```

```cmake
set(DEPS_INSTALL_DIR "${CMAKE_BINARY_DIR}/usr")
```
